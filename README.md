# Документация проекта "Клубничный Король"

## Описание проекта

"Клубничный Король" - это браузерная RPG-игра, симулятор жизни самозанятого предпринимателя. Игрок принимает решения, которые влияют на три основных параметра: деньги, энергия и репутация.

## Структура проекта

### HTML страницы

#### 1. `index.html` - Главная страница
- **Назначение**: Стартовая страница игры
- **Элементы**:
  - Логотип игры (`logo.png`)
  - Корона (`crown.png`)
  - Описание игры в желтом блоке
  - Кнопка "Ю-ху" для начала игры
- **Переход**: При нажатии на кнопку происходит переход на `comics.html`

#### 2. `comics.html` - Страница с комиксами
- **Назначение**: Вводная история в виде комиксов перед началом игры
- **Элементы**:
  - 9 кадров комикса (переключаются кнопкой "Далее")
  - Кнопка навигации
- **Используемые изображения**:
  - `Comics_01.png`
  - `Comics_02.png`
  - `Comics_03.png`
  - `Comics_04.png`
  - `Comics_05.png`
  - `Comics_06.png`
  - `Comics_07.png`
  - `Comics_08.png`
  - `Comics_09.png`
- **Переход**: После просмотра всех комиксов (на 9-м кадре) кнопка меняется на "вот дела!" и ведет на `onboarding.html`

#### 3. `onboarding.html` - Страница обучения
- **Назначение**: Обучение игрока перед началом игры
- **Элементы**:
  - Верхнее изображение с инструкцией (`onboarding_top.png`)
  - Три изображения с инструкциями:
    - `onboarding_instruction_01.png`
    - `onboarding_instruction_02.png`
    - `onboarding_instruction_03.png`
  - Изображение панели игрока (`onboarding_player_bar.png`)
  - Кнопка "Ю-ХУ-ХУ!" для начала игры
- **Переход**: При нажатии на кнопку происходит переход на `game.html`

#### 4. `game.html` - Основная игровая страница
- **Назначение**: Главная игровая механика
- **Элементы**:
  - Счетчик шагов с таймером
  - Изображение сцены (из папки `asset/scenes/`)
  - Текстовая информация о текущем шаге
  - Три варианта выбора (A, B, C)
  - Панель игрока с:
    - Аватаром (меняется в зависимости от уровня энергии)
    - Счетчиком денег
    - Счетчиком энергии
    - Счетчиком репутации
  - Экран результата после выбора
  - Экран окончания игры
  - Экран проигрыша (выгорание)
- **Используемые изображения**:
  - Фон: `strawberry_pattern.png` (анимированный фон)
  - Счетчик шагов: `steps_count_back.png`
  - Иконки ресурсов:
    - `money_icon.png` - деньги
    - `energy_icon.png` - энергия
    - `reputation_icon.png` - репутация
  - Аватары игрока (меняются в зависимости от энергии):
    - `player_ava_-4.png` (энергия < 5)
    - `player_ava_-3.png` (энергия 6-10)
    - `player_ava_-2.png` (энергия 11-20)
    - `player_ava_-1.png` (энергия 21-30)
    - `player_ava_0.png` (энергия 31-50)
    - `player_ava_+1.png` (энергия 51-60)
    - `player_ava_+2.png` (энергия 61-70)
    - `player_ava_+3.png` (энергия 71-80)
    - `player_ava_+4.png` (энергия > 81)
    - `player_ava.png` (базовый, используется по умолчанию)

## CSV файлы

### `GameParsing.csv` - Основной файл данных игры

**Расположение**: Корневая папка проекта (`GameParsing.csv`)

**Формат**: CSV файл с заголовками (header)

**Структура колонок**:

| Колонка | Описание |
|---------|----------|
| `step` | Номер шага игры (1-10) |
| `header` | Заголовок шага (отображается в верхней части экрана) |
| `text` | Основной текст описания ситуации |
| `image_url` | Путь к изображению сцены (например, `asset/scenes/1step.png`) |
| `vA_header` | Заголовок варианта выбора A |
| `vA_text` | Краткое описание варианта выбора A |
| `vA_result` | Текст результата после выбора варианта A |
| `vA_m` | Изменение денег при выборе варианта A (может быть положительным или отрицательным) |
| `vA_e` | Изменение энергии при выборе варианта A (может быть положительным или отрицательным) |
| `vA_r` | Изменение репутации при выборе варианта A (может быть положительным или отрицательным) |
| `vB_header` | Заголовок варианта выбора B |
| `vB_text` | Краткое описание варианта выбора B |
| `vB_result` | Текст результата после выбора варианта B |
| `vB_m` | Изменение денег при выборе варианта B |
| `vB_e` | Изменение энергии при выборе варианта B |
| `vB_r` | Изменение репутации при выборе варианта B |
| `vC_header` | Заголовок варианта выбора C |
| `vC_text` | Краткое описание варианта выбора C |
| `vC_result` | Текст результата после выбора варианта C |
| `vC_m` | Изменение денег при выборе варианта C |
| `vC_e` | Изменение энергии при выборе варианта C |
| `vC_r` | Изменение репутации при выборе варианта C |

**Пример строки**:
```csv
1,Квартира. Жена против. Первое решение,Кидаешь клубнику в шоколаде на стол. Жена в ярости: Бюджет не на твои идеи!,asset/scenes/1step.png,Отложить разговор до утра.,,"Утро. Идея на месте. Видимо, отложить можно разговор, но не мысли",+0,-5,+10,...
```

**Примечания**:
- Каждая строка представляет один шаг игры
- Шаги должны быть отсортированы по номеру (`step`)
- Значения изменений ресурсов могут быть положительными (например, `+10`) или отрицательными (например, `-5`)
- Путь к изображению должен быть относительным от корня проекта
- Текст может содержать переносы строк, которые будут преобразованы в HTML (`<br>`)

## Игровая механика

### Начальные параметры:
- **Деньги**: 50
- **Энергия**: 50
- **Репутация**: 0

### Условия окончания игры:
1. **Выгорание**: Если энергия падает до 0 или ниже - игра заканчивается экраном проигрыша
2. **Завершение**: После прохождения всех 10 шагов показывается финальный экран с итоговой статистикой

### Таймер:
- Время на принятие решения зависит от уровня энергии:
  - Энергия > 75: 36 секунд
  - Энергия ≥ 25: 27 секунд
  - Энергия < 25: 18 секунд
- Если время истекает, выбирается случайный вариант (A, B или C)

### Визуальные изменения:
- Аватар игрока меняется в зависимости от уровня энергии (см. список аватаров выше)
- Анимация фона с паттерном клубники на всех страницах
